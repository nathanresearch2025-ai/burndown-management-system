## 🎯 性能分析规则

### 吞吐量评估
- RPS提升 > 50%: ✅ 优秀
- RPS提升 20-50%: ⚠️ 一般
- RPS提升 < 20%: ❌ 较差

### 稳定性评估
- 成功率 ≥ 99.9%: ✅ 优秀
- 成功率 ≥ 99%: ✅ 良好
- 成功率 < 99%: ⚠️ 需改进

### 响应时间评估
- 增长 < 100%: ✅ 稳定
- 增长 100-300%: ⚠️ 下降
- 增长 > 300%: ❌ 严重下降

### P95响应时间评估
- < 100ms: ✅ 优秀
- 100-500ms: ✅ 良好
- > 500ms: ⚠️ 需优化

## 🐛 故障排查

### 问题1: 图表显示方框
**原因**: matplotlib中文字体问题
**解决**: 已改用英文标题，不影响使用

### 问题2: 汇总报告未更新
**原因**: generate_summary.py执行失败
**解决**: 手动运行 `python3 /myapp/test/pressure/generate_summary.py`

### 问题3: 性能分析不准确
**原因**: 测试时长过短或并发数过低
**解决**: 使用默认配置（baseline 5用户/30秒，standard 20用户/60秒，peak 50用户/60秒）

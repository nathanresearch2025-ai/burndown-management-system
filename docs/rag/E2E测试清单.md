# AI 任务生成功能 - 端到端测试清单

## 测试环境准备

### 1. 环境配置

- [ ] PostgreSQL 数据库已启动并初始化
- [ ] Redis 服务已启动
- [ ] 后端应用已启动（端口 8080）
- [ ] 前端应用已启动（端口 5173）
- [ ] AI 配置已正确设置（或使用降级模式测试）

### 2. 测试数据准备

- [ ] 至少创建 1 个项目
- [ ] 至少创建 1 个活跃冲刺
- [ ] 至少创建 5-10 个历史任务（用于相似度匹配）
- [ ] 至少创建 1 个测试用户并登录

# 全局快速访问菜单功能

## 功能概述

在系统右上角添加了"快速操作"按钮，提供全局快速访问功能，用户可以在任何页面快速创建项目、Sprint、任务，或快速导航到常用页面。

## 主要特性

### 1. 快速创建功能
- **新建项目**：在任何页面快速创建新项目
- **新建Sprint**：快速创建Sprint（支持选择所属项目）
- **新建任务**：快速创建任务（支持选择项目和Sprint）

### 2. 快速导航
- **我的待办**：跳转到仪表盘
- **所有项目概览**：跳转到项目列表页
- **报表中心**：跳转到仪表盘（未来可扩展独立报表页）

### 3. 帮助文档
- **帮助 & 文档**：打开帮助文档（新标签页）

### 4. 键盘快捷键
- 按 `Ctrl + /` 可快速呼出/关闭快速访问菜单

## 技术实现

### 新增组件

1. **MainLayout** (`src/components/Layout/MainLayout.tsx`)
   - 统一的页面布局组件
   - 包含顶部导航栏和快速访问菜单
   - 所有页面统一使用此布局

2. **QuickAccessMenu** (`src/components/Layout/QuickAccessMenu.tsx`)
   - 快速访问下拉菜单组件
   - 支持键盘快捷键（Ctrl + /）
   - 分组显示：创建类、导航类、其他

3. **可复用Modal组件**
   - `CreateProjectModal` - 创建项目弹窗
   - `CreateSprintModal` - 创建Sprint弹窗（支持项目选择）
   - `CreateTaskModal` - 创建任务弹窗（支持项目和Sprint选择）

### 更新的页面

所有主要页面已更新为使用统一的 `MainLayout`：
- Dashboard
- ProjectList
- SprintBoard
- TaskBoard
- BurndownChart

## 使用说明

### 快速创建

1. 点击右上角"快速操作"按钮
2. 选择要创建的对象类型
3. 在弹出的表单中填写信息
4. 点击确定完成创建

### 键盘快捷键

- 在任何页面按 `Ctrl + /` 打开快速访问菜单
- 再次按 `Ctrl + /` 关闭菜单

### 智能提示

- 创建Sprint时，如果未选择项目，会提示先选择项目
- 创建任务时，需要先选择项目，然后才能选择该项目下的Sprint

## 未来扩展

根据PRD中的P2优先级功能，未来可以扩展：

1. **系统设置**：添加管理员权限检查，仅管理员可见
2. **报表中心**：创建独立的报表聚合页面
3. **我的待办**：创建个人待办列表页面
4. **自定义菜单**：支持管理员自定义菜单项（插件化）
5. **移动端适配**：底部浮动按钮或汉堡菜单

## 性能优化

- 菜单响应时间 < 100ms
- 使用 React Query 缓存项目和Sprint数据
- Modal 组件按需加载，使用 `destroyOnClose` 优化内存

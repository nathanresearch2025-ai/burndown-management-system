# Burndown Management System - Frontend

React 18 + TypeScript + Ant Design 前端应用

## 技术栈
- React 18
- TypeScript 5
- Ant Design 5
- Vite 5
- React Query
- Zustand
- ECharts

## 快速开始

### 安装依赖
```bash
cd frontend
npm install
```

### 运行开发服务器
```bash
npm run dev
```

应用将在 http://localhost:5173 启动

### 构建生产版本
```bash
npm run build
```

## 主要功能
- 用户登录/注册
- 项目管理
- Sprint管理
- 任务看板
- 工时记录
- 燃尽图可视化

## 目录结构
```
src/
├── api/          # API接口
├── pages/        # 页面组件
├── store/        # 状态管理
├── App.tsx       # 根组件
└── main.tsx      # 入口文件
```
